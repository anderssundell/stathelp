Welcome to the Statistics help!

Here you can find simple guides to different techniques for statistical analysis. The focus is on running and interpreting the analyses, not the theory and assumptions behind that underpin the analyses. In the guides the code and output from the statistics software is shown, together with explanations in text. All code is supposed to be reproducible, so if you want to you can download the data that is linked in the guides, and follow along in the instructions. The site is run by Anders Sundell.
